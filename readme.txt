== Description ==

This plugin adds the following features to your website:

- Adds placeholders to a page or post for dynamic content
- Adds the possibility of defining several contents for each placeholder
- The content to be shown on the web page will be decided by machine learning algorithms in real-time adapted to who is browsing (on page load)
- Adds a dashboard to visualize Click Through Rate (CTR) results on dynamic content that measures the effectiveness of the machine learning algorithms
